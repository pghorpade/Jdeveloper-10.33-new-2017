Standard-1.0 Taglib
JSP(tm) Standard Tag Library 1.0 implementation

Jakarta Taglibs hosts the Standard Taglib 1.0, an implementation of the JSP Standard Tag Library (JSTL), version 1.0, which was developed under the Java Community Process.

NOTE: Standard-1.0 (JSTL 1.0) requires a JSP container that supports the Java Servlet 2.3 and JavaServer Pages 1.2 specifications. Jakarta Tomcat 4 supports these specifications. The Standard 1.0 taglib has been tested with Tomcat 4.1.24.

Standard-1.1 (implementation of the JSTL 1.1 specification) requires a JSP container that supports the Java Servlet 2.4 and JavaServer Pages 2.0 specifications. Jakarta Tomcat 5 supports the new specifications. The Standard-1.1 taglib has been tested with Tomcat 5.0.3.
