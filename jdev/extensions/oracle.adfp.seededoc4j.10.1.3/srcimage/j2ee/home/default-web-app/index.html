Preconfigured Portlet Producers

This server contains the following preconfigured portlet producers:

WSRP Portlet Producers

  • Rich Text Portlet Producer
  • Sample Portlets

PortalTools Portlet Producers

  • PortalTools Welcome Page
  • OmniPortlet Producer
    Note: To access data through a firewall, you need to configure a proxy by using the <proxyInfo> tag in OmniPortlet's provider.xml file. For more information refer to the Oracle WebCenter Framework Developer's Guide.
  • Web Clipping Portlet Producer
    Note: To access data through a firewall, you need to configure a proxy by using the <proxyInfo> tag in Web Clipping's provider.xml file. For more information refer to the Oracle WebCenter Framework Developer's Guide.

PDK-Java Portlet producers

  • PDK-Java Sample Producer
  • PDK-Java Struts Sample Producer
