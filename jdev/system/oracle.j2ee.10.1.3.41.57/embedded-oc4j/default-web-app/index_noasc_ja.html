概要

Oracle Containers for J2EE 10g （10.1.3.3.0）は、Oracle Application Serverの主要なJ2EEランタイム・コンポーネントです。

今回のリリースには、次の特長があります。

  • EJB 3.0 Early Draft仕様に対応したJ2EE 1.4 Compatible
  • JAX-RPC、WS-I、WS-Security、WS-Reliabilityに対応した拡張Webサービス機能
  • Oracle TopLinkによる業界トップのオブジェクト・リレーショナル永続化ソリューション
  • Enterprise Manager Application Server Controlによる標準ベースの管理および配置サポート
  • エンタープライズ・パフォーマンスとスケーラビリティ
ドキュメント

 

今回の配布には、構成および管理ガイドが含まれています。

  • サーバーの管理に役立つ構成および管理ガイド

OC4Jの管理

 

OC4Jは、Application Server Controlを使用してブラウザで管理します。

このインスタンスは、リモートで管理するように構成されます。
